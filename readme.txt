Show a sitewide notification bar.

== Description ==

This plugin allows you to show a notification bar across the top of your site.	

== Installation ==

1. Upload the `hotplates` directory to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
